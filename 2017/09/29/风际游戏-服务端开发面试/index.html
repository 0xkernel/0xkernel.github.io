前言

    前天我们学校有个“天府人才行动2017秋季校园行”（通俗的讲就是校园招聘）。由于现在用 C++ 的公司少之又少，于是我抱着去见见世面的心态去参加，就带了4份简历（我那时觉得我简历排版已经很好了，然而昨天就被我基友给完爆了）过去，还担心一份都投不出去（哎，不得不说现在C++的工作好少，基本上都是java的）。开始去了一个招聘游戏服务端开发的岗位，把简历递了过去，然而人家看了看我简历，说，我们服务端用的是Java，你会Java么？然后我就尴尬的走了。
    我在看了看周围的公司，发现风际游戏招聘服务端开发，这次我就没有直接递简历，而是先问了问那个招聘的小姐姐，你们招聘的服务端开发是用c++的么？然后小姐姐说是，我就把我简历递上去了（我说了句，哇，有c++服务端开发的，小姐姐然后说：哈哈，看你一听c++服务端开发的，就高兴的递简历给我，是不是其他公司用的java啊，然后我就点了点头），然后小姐姐就看我简历，问了一些简历上面的项目，最后问了问单列模式的优缺点，然后给了我一张笔试地点的小纸条，最后给我说，我挺看好你的。
    笔试时间是下午2点半，题目都是一些挺基础简单的东西。最后有个编程题是swap list node，leetcode上有。然后我晚上6点左右就收到了面试邀请。就今天去风际游戏公司参加面试了。
    公司就在天府软件园，环境挺不错的。

一面

    到了公司，去了一个小型会议室那种房间里，然后等了一会儿，一个hr小姐姐就叫去我去另一个小房间里等待面试，然后我在那房间里等了会儿，一个和蔼的面试官就进来了，然后叫我做自我介绍(插曲:上次去四川大学望江校区面试一个实验室的一个开发岗位(get offer),那个面试官叫我自我介绍,我就直接照着简历上说,然后那个面试官就直接打住我,说简历上已经写了的你就不说,你就说说你平时的爱好,不用太紧张.不得不说,我挺感谢那面试官的指点,让我知道什么叫”自我介绍”,因为这样自我介绍后,我面前这个面试管看了我,很和蔼的笑了笑),
    开始问我你最精通那种语音,然后我一脸蒙蔽,回了句”精通?”(内心OS:我特么敢说我精通某某语言,我就是找死啊),
    面试官:”你觉得你对c++掌握的怎么样”,我:”一般吧”(内心OS:我特么敢说我熟悉,精通c++就是找虐,你还是慢慢来探索我的掌握度吧)
    然后面试官就出了两道c++题,一道是内存对齐,一道是const ( const int func(const int a)const,这3个const的作用 ,接着问const成员函数,如何修改对象数据,我没答上来(mutable修饰)),const那道没答好.
    然后问了问我简历上的一些项目，然后tcp与udp的区别，最后是reactor模式.就叫我等一会儿.因为const那道题没答好,我就觉得我已经GG了,就把桌上的简历收进包里,等hr小姐姐通知我,让我走人.
    然而万万没想到的是,我还有下一个面试官.

二面

    另一个面试官(后面知道是那家公司的CTO)就进来了,也挺和蔼,亲人的那种,且一眼就可以看出是那种技术特牛逼的那种(有个小细节不得不为这个面试官点赞,就一面完之后,我趴在房间里窗口处的护栏旁,纠结我是坐公交回去,还是起单车回去,然后面试官就来了,因为我是背对着门,没看见面试官来了,面试官就敲了敲门,然后我就回过神,面试官朝我笑笑点点头进来了).
    然后还是面试老套路,叫我自我介绍.自我介绍完了之后,就问我简历上的一些项目.Reactor模式.
    面试官:”哇,你居然学过逆向工程”,我:”对”,(内心OS:哇,居然有面试官注意到这个,这个面试官貌似还学过这玩意).
    然后问我,”你是自学的么”.我:”对”.
    面试官:”你学一门新技术的话,你一般会怎么做”,我:”先去网上搜说对应技术的一些推荐书籍,然后在去豆瓣上看看评论,适不适合我读”.
    面试官:”那你对以后的发展有什么计划没?”,我:”学习Go语言,然后学习分布式系统,在有业余的时间情况下学习Linux内核.然后找一个公司,学习业务编程.”
    然后面试官问了我一些游戏的东西(不得不说还好我玩游戏,还好我初中,高中的时候搭建过私服.)
    面试官:”你看过特南鲍姆的书么?”,我:”看过,<<现代操作系统>>,<<计算机网络>>,<<计算机组成结构化>>,然后现在准备看他那本分布式系统的书”,(内心OS:哈哈,还好特南鲍姆是我最喜欢的一个计算机科学家了,他那几本书我看过三本)
    面试官:”你们学校是不是基本上都是学Java的”,我:”嗯”,面试官:”那你是你们学校的另类了”,我:”蛤?”,(内心OS:喵喵喵?褒义还是贬义?)
    然后面试官问我,你有问题问我没?我就问了问服务器负载均衡.(二面有些细节记不起来了,且有些东西太高深了,我也听不懂,反正这个面试官思路特别清洗,技术特别牛)
    然后面试就叫我去外面等结果.我就很自觉的来到公司的电梯哪里等待叫我滚蛋.

hr面

    等了几分钟,然后hr小姐姐把我叫到另一个小房间,给我说你技术面过了.然后hr面基本没什么,就是谈薪,问多久来到岗.最后给了个口头offer,叫我做明天的智力测试与心里测试(智力测试我有点慌,怕是过不了哦).

总结

    面试官没问数据结构与算法,有点失望,虽然我掌握的也就一般.然后有时候有些东西没有表达清楚,看来得多练练表达能力了,就从写博客开始吧!
